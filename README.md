# BearLang Tutor

BearLang Tutor — это небольшое консольное приложение на C++, которое помогает детям учиться программированию.  
Вы вводите программу на языке BearLang, приложение переводит её в эквивалентный C++-код, показывает результат перевода
и сразу же компилирует/запускает его, чтобы увидеть живой вывод.

## Сборка и запуск

Требуются `cmake` (3.8+) и любой компилятор с поддержкой C++20 (например, `g++`/`clang++`).

```bash
cd app
cmake -S . -B build
cmake --build build
./build/app        # Windows: .\build\Debug\app.exe
```

> Примечание. Во время исполнения переведённый файл компилируется отдельной командой `g++`. Убедитесь, что компилятор доступен в `PATH`.

## Как пользоваться

1. Запустите программу и выберите пункт «Написать и выполнить программу BearLang».
2. Вводите код построчно. Когда закончите, напечатайте `конец` на отдельной строке.
3. Приложение покажет сгенерированный C++ и спросит ввод/вывод при необходимости (через тот же терминал).

### Пример BearLang

```
целое число = 5
целое попытка = 0
пока (попытка < число)
    вывод "Привет, BearLang!"
    попытка = попытка + 1
если (число >= 5)
    вывод "Число достаточно большое"
иначе
    вывод "Нужно побольше"
```

Пример выше превратится в полноценный `main` на C++ и выведет приветствие пять раз, затем одно из сообщений условия.

## Памятка по синтаксису BearLang

- **Типы**: `целое` (int), `дробное` (double), `строка` (std::string), `логика` (bool, значения `правда`/`ложь`).
- **Ввод/вывод**: `ввод a` (считывает из stdin), `вывод выражение` (печатает и переводит строку).
- **Условия**: `если (...)`, `иначе если (...)`, `иначе`. Логические операторы `и`, `или`, `не`, сравнения `< > <= >= ==`.
- **Циклы**: `пока (условие)` и `для (целое i от 0 до 10)` — второй вариант идёт по включительному диапазону и добавляет `++i`.
- **Арифметика**: поддерживаются `+ - * / % ^`. Оператор `^` конвертируется в `std::pow(...)`, для сложных выражений рекомендуется использовать скобки вокруг операндов.

## Как это работает

- Парсер читает программу построчно (отступы определяют блоки), строит промежуточное представление и генерирует C++.
- Сгенерированный файл временно сохраняется в каталоге `std::filesystem::temp_directory_path`, компилируется командой `g++ -std=c++20`, затем запускается.
- После завершения временные файлы удаляются, поэтому можно безопасно запускать приложение много раз.

Экспериментируйте с BearLang, пробуйте разные конструкции и смотрите, во что они превращаются на C++!
